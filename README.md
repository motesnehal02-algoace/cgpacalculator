# CGPA Calculator

A beginner-friendly C++ project that calculates Semester GPA and Overall CGPA using course grades and credit hours.

## Features

- Accepts multiple courses as input
- Calculates total credits
- Calculates semester GPA
- Calculates overall CGPA
- Displays individual course details

## Technologies Used

- C++
- Arrays
- Loops
- Switch Statements
- Conditional Statements
